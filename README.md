# Search-and-Optimization-Algorithims
Revision for Intelligent Systems lab test.

**The Following Algorithims are discussed:**

**Search:** BFS, DFS, IDDFS, DLS, UCS, GBFS, A*, Minimax, and Alph-Beta-Pruning

**Optimization:** Simple HC, Steepest-Ascent HC, RR HC, Stochastic HC, Simulated Annealing

**First Step: Create a Virtual Environment**

You must have anaconda installed.

1- Open Anaconda Prompt

2- Create vitual environment
    
    conda create -n myenv python=3.9

3- Activate the environment
    
    conda activate myenv

4- Install pyamaze
   
    pip install pyamaze

5- In VS code, press Ctrl+Shift+P, then look for select inerpreter and select myenv.
